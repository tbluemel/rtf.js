---- Windows 95 and NT -----
MS-IME for Japanese
Changing input mode
There are six different input modes when you activate the IME. Those are;
Double-byte Hiragana input mode （全角ひらがな）
Double-byte Katakana input mode （全角カタカナ）
D ouble-byte Roman character input mode （全角英数）
Single-byte Katakana input mode （半角カタカナ）
Single-byte Roman character input mode （半角英数）
Direct input (Single-byte) mode （直接入力）
You can change the mode from MS-IME toolbar.
